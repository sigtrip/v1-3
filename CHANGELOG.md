# Changelog

Все заметные изменения проекта фиксируются в этом файле.

## v1.3.1 — 2026-03-02

### Added
- Веб-панели (`web_dashboard` и `fastapi_dashboard`) получили кнопку `🎙 СЛУШАТЬ` с браузерным SpeechRecognition для голосового ввода команд.
- Desktop GUI расширен новыми режимами выбора ИИ: `Watsonx`, `OpenAI`, `Grok`.

### Changed
- `src/core.py`: добавлена нормализация placeholder-секретов из `.env`, чтобы фейковые/шаблонные значения не считались валидной конфигурацией провайдеров.
- `src/core.py`: улучшена инициализация провайдеров (`Gemini`, `GigaChat`, `YandexGPT`, `OpenAI`, `Grok`, `Watsonx`) и добавлен флаг `GIGACHAT_VERIFY_SSL`.
- `src/connectivity/telegram_bot.py`: усилена валидация `TELEGRAM_BOT_TOKEN`/`USER_ID` и стабилизирован запуск polling в threaded-режиме (`run_polling(stop_signals=None)`).
- `requirements-optional.txt`: убрана жёсткая pip-зависимость `SoapySDR>=0.8.1`, добавлены кроссплатформенные инструкции системной установки.

### Fixed
- Устранён runtime-сбой Telegram-моста (`set_wakeup_fd only works in main thread`) при запуске бота в отдельном потоке.
- Снижен шум конфигурационных ошибок при placeholder-значениях API ключей.
- Исправлена доступность голосового ввода в веб-интерфейсах.

## v1.3.0 — 2026-03-01

### Added
- Квантовый гомеостаз железа: мониторинг CPU/RAM/температуры и автоматические защитные состояния `Protective`/`Unstable`.
- Автономное любопытство v2: idle-исследование фактов из памяти, веб-поиск и сохранение инсайтов в SQLite/память.
- Smart P2P-маршрутизация: классификация нод (`gateway`/`worker`/`server`) и приоритетная отправка тяжёлых задач на более сильные ноды.
- Smart Flasher: USB-детект чипов (ESP32/RP2040/STM32), генерация заготовки прошивки и автопопытка заливки через `esptool`/`PlatformIO`.
- GitOps-модуль для встроенного управления репозиторием (`status`/`commit`/`push`) из команд Аргоса.

### Changed
- `src/quantum/logic.py`: добавлены внешняя телеметрия, thermal-риск и возможность временного принудительного состояния.
- `src/core.py`: интеграция подсистем homeostasis/curiosity/gitops, новые интенты и ограничения тяжёлых задач при перегреве.
- `src/connectivity/p2p_bridge.py`: усилен диспетчер задач, исправлен удалённый роутинг через TCP JSON-протокол.
- `src/skills/evolution/skill.py`: жёсткий промпт-каркас, валидация исполняемого кода и anti-markdown gate.
- `README.md`: документация обновлена под новые возможности и команды.

### Notes
- Тег релиза: `v1.3.0`.
- Совместимость: без breaking changes в базовых сценариях запуска.
