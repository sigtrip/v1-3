# Argos Prompt Examples

Набор базовых промптов для ручных проверок и демо.

## System / Health

- `статус системы`
- `чек-ап`
- `iot статус`

## Memory / RAG

- `запомни имя: Всеволод`
- `найди в памяти имя`
- `граф знаний`

## Vision / Smart

- `посмотри на экран что происходит в интерфейсе`
- `умные системы`
- `статус устройства sensor_01`

## P2P / Network

- `запусти p2p`
- `статус сети`
- `распредели задачу проверь температуру узлов`

## Tool Calling

- `какая погода и сколько свободно места на диске?`
- `json схемы инструментов`
