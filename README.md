# 🔱 ARGOS UNIVERSAL OS v1.31-Singularity

## 🌌 Overview
Argos is a high-performance, autonomous AI orchestrator designed for Colab, Desktop, and Android environments. It integrates quantum logic states, persistent memory, and an evolutionary skill-creation system.

## 🚀 Core Features v1.31
- **NeuralNexus (Local ML)**: A self-learning classifier trained on local SQLite chat history. Use `модель обучить` to sync its brain.
- **Quantum Logic Engine**: 5 distinct operational states (Analytic, Creative, Protective, Unstable, All-Seeing) that shift based on system load and context.
- **PyPI Orchard**: Automatic packaging and publishing of generated skills to the Python Package Index.
- **Persistent Memory**: SQLite-backed history tracking with semantic recall capabilities.
- **GhostLink C2**: Hidden communication channels for remote directive broadcasting.

## 🛠️ CLI Commands
- `status` - Full system diagnostic report
- `модель статус / обучить / сохранить` - Manage the local ML core
- `pypi список / опубликовать [skill]` - Skill distribution management
- `создай навык [описание]` - AI-driven code generation and dna integration
- `запомни [ключ]=[значение]` - Direct fact injection into long-term memory

## 📦 Deployment
1. Fill `.env` with required API keys (Gemini, Telegram, OpenAI).
2. Run `pip install -r requirements.txt`
3. Execute `python main.py --no-gui`
