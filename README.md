# 👁️ ARGOS UNIVERSAL OS (v1.0.0-Absolute)

> **"Аргос — всевидящий, всезнающий, неизменный."**

**ARGOS** — это децентрализованная автономная ИИ-система с квантовым управлением состояниями, предназначенная для глубокого мониторинга, системного администрирования и расширения возможностей Создателя через симбиоз локальных и облачных вычислений.

---

## 🌌 Архитектурные Особенности

* **🧠 Гибридный Интеллект:** Бесшовное переключение между облачным ядром (Google Gemini 1.5 Pro) и локальным (Ollama/Llama 3) при потере связи.
* **⚛️ Квантовая Логика:** Принятие решений и выбор тона общения на основе 5-позиционной квантовой симуляции вероятностей.
* **🛠️ Deep Admin:** Прямой доступ к управлению процессами, файловой системой и терминалом операционной системы.
* **📡 Сетевые Очи (Argos Eyes):** Анонимный скрапинг глобальной сети через DuckDuckGo для получения данных в реальном времени.
* **🛡️ Протокол «Щит»:** Шифрование всех данных по стандарту AES-256 и автоматическая проверка безопасности репозитория (GitGuard).
* **📱 Удаленный Мост:** Полноценное управление системой через защищенный Telegram-канал.

## 📂 Структура Системы

```text
ArgosUniversal/
├── main.py                          # Оркестратор (Запуск системы)
├── genesis.py                       # Первичная инициализация
├── requirements.txt                 # Зависимости Python
├── Dockerfile                       # Контейнеризация
├── docker-compose.yml               # Docker Compose конфигурация
├── src/
│   ├── core.py                      # Диспетчер команд и логика ИИ
│   ├── admin.py                     # Управление файлами и процессами
│   ├── evolution.py                 # Модуль самозаписи кода
│   ├── quantum/logic.py             # Квантовый генератор состояний
│   ├── security/                    # Шифрование и защита ключей
│   │   └── encryption.py            # AES-256 и GitGuard
│   ├── factory/                     # Прошивка IoT и репликация системы
│   │   ├── flasher.py               # Сканирование портов и устройств
│   │   └── replicator.py            # Создание архивов системы
│   ├── skills/web_scrapper.py       # Сетевой мониторинг
│   └── connectivity/                # Telegram, Геолокация, P2P
│       └── telegram.py              # Telegram мост
└── config/identity.json             # Файл личности Аргоса
```

## 🛠 Установка и Развертывание

**1. Клонирование репозитория:**
```bash
git clone https://github.com/sigtrip/v1-3.git
cd v1-3
```

**2. Развертывание окружения:**
```bash
python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate      # Windows
pip install -r requirements.txt
```

**3. Первичная инициализация (Genesis):**
```bash
python genesis.py
```

## ⌨️ Основные команды управления

| Команда | Описание | Модуль |
|---|---|---|
| `статус системы` | Вывод данных о ЦП, ОЗУ и ОС | admin.py |
| `файлы [путь]` | Просмотр содержимого директории | admin.py |
| `убей процесс [имя]` | Принудительное завершение процесса | admin.py |
| `консоль [команда]` | Выполнение системной команды | admin.py |
| `сканируй порты` | Поиск подключенных устройств/контроллеров | flasher.py |
| `репликация` | Создание полной архивной копии системы | replicator.py |
| `квантовое состояние` | Проверка квантового вектора | quantum/logic.py |
| `аудит безопасности` | Проверка безопасности config | security/encryption.py |
| `поиск [запрос]` | Поиск в сети через DuckDuckGo | skills/web_scrapper.py |
| `новости` | Получение технологических новостей | skills/web_scrapper.py |
| `помощь` | Список всех доступных команд | core.py |
